Explorando Tags Básicas

1. HTML: Define o início e o fim de um documento HTML

2. HEAD: Contém metadados e informações sobre a página (título, links de CSS, scripts,etc.)

3. TITLE: Define o título da página exibido na aba do navegador

4. BODY: Contém todo o contúdo visível da página (textos, imagens, links, etc)

5. H1 a H6: Títulos e subtítulos, sendo h1 o mais importante e h6 o menos importante

6. P: Define um parágrafo de texto

7. A: Cria um link para outra página ou recurso

8. IMG: Exibe uma imagem

9. UL: Lista não ordenada (com marcadores)

10. TABLE: Cria uma tabela para organizar dados
